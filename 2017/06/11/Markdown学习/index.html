发表于 2017-06-11 |

今天用了两个小时时间熟悉了一下Markdown的基本语法

一级标题

#一级标题

二级标题

##二级标题

三级标题

###三级标题

四级标题

####四级标题

以此类推，Markdown总共有6级标题，分别在行首插入1到6个#

代码区块的使用，每行锁进4个空格或者1个制表符

在代码区块中“<、>、&”会自动被转换成HTML实体，可以在代码区块
中很容易插入HTML原始代码

分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，
行内不能有其他东西。你也可以在星号或是减号中间插入空格。
下面每种写法都可以建立分隔线

* * *
- - - 
_ _ _

链接

行内式

This is an example inline link.

This link has no title attribute.

This is [an example](http://llmrrongll.github.io     "Title") inline link.

[This link](http://llmrrong.github.io) has no title     attribute.

参考式

This is an example reference-style link.

强调

使用*和_作为标记强调字符的符号，被*或_包围的字符会转成 <em>包围

*需要强调的字符*

需要强调的字符

代码

小段代码用反引号

NSLog(@"test");

大段的代码在行首缩紧四个空格或者一个制表符即可

图片

使用类似链接的语法

![Alt text] (img path)

自动链接

使用尖括号包起来就会自动转成链接

<http://llmrrongll.github.io>

http://llmrrongll.github.io

反斜杠

Markdown中的反斜杠的作用类似于Objective-C中的转义字符，可以插入一些特殊符号， Markdown支持以下这些符号前面加上反斜杠来插入普通字符
