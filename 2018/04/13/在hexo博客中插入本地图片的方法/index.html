在hexo博客中插入本地图片的方法

发表于 2018-04-13 |

使用hexo搭建博客，生成博文之后文章中的图片却只能加载网络图片？
为什么不能加载hexo网站目录内的图片呢？于是乎…某度之后，有了本文，
这位同学Nuub的博客很好的解决了我的这个问题

整体步骤总结如下：

  1. 打开博客网站根目录下的配置文件_config.yml，并修改
    post_asset-folder: true

  1. 在终端cd到博客目录下，运行npm install hexo-asset-image –save安装插件

  2. 执行hexo n xxxx之后会在source/_post目录下生成名为xxxx.md和一个同名的文件夹，之后要插入文章的图片就是要放在这个同名文件夹中

  3. 在需要引入图片的地方使用如下方法

    ![图片替换文字](文章名称/图片名称.png)

  4. 最后检查一下，hexo g生成页面后，进入public\2017\02\26\index.html文件中查看相关字段，可以发现，html标签内的语句是<img src="2017/02/26/xxxx/图片名.jpg">，而不是<img src="xxxx/图片名.jpg>。这很重要，关乎你的网页是否可以真正加载你想插入的图片。

总结结束。
